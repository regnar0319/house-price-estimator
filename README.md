# house-price-estimator
A Multilingual Geospatial ML Dashboard

This project predicts residential property prices globally using Latitude/Longitude and structural features (Area, Garage, etc.).

Key Features:

Geospatial Intelligence: Interactive map-based location selection.

Global Reach: Full UI support for 15 languages (Hindi, Marathi, Spanish, etc.).

Advanced ML: Powered by an XGBoost model trained on geospatial datasets.


How to Run:

1.Clone the repo.

2.Double-click run_app.bat (Windows) or run streamlit run app.py.


Development Environment: Built using Google Antigravity (Agent-First IDE).

Workflow Method: Used Planning Mode for architectural design and Mission Mode for multi-agent execution.

Infrastructure: Automated deployment using run_app.bat and managed virtual environments with agent-assisted scripts.
